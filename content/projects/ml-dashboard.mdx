## 项目概述

机器学习实验仪表板是一个专为 ML 研究团队设计的实验跟踪和可视化平台。

## 核心功能

- **实验跟踪**: 记录所有超参数和指标
- **实时可视化**: 训练过程中的实时曲线更新
- **模型比较**: 并排比较不同实验的的结果
- **团队协作**: 多用户支持和权限管理
- **自动报告**: 生成专业的实验报告
- **API 集成**: 与 PyTorch 和 TensorFlow 无缝集成

## 技术架构

### 前端
- **Next.js 15**: React 框架，支持 SSR
- **TypeScript**: 类型安全
- **Recharts**: 数据可视化
- **Tailwind CSS**: 样式系统

### 后端
- **Next.js API Routes**: 服务端 API
- **PostgreSQL**: 数据库
- **Prisma**: ORM
- **NextAuth.js**: 身份验证
